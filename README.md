# watchOut
## 시각장애인용 웨어러블형 길안내 어플  

와치아웃은 시각장애인의 자립과 성취를 위한 웨어러블형 길찾기 서비스입니다.  
시각정보 없이도 안전하게 길을 찾을 수 있도록 ui와 음성을 최소화 하고 모든 길 안내를 진동으로 구현하였습니다.  
와치아웃은 '빠른 길'이 아닌 '안전한 길'을 우선 추천합니다. 이 때, '안전한길' 의 기준이 되는 대상은 비장애인이 아닌 시각장애인입니다.
